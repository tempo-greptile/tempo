# Tempo Improvement Proposals (TIPs)

Tempo Improvement Proposals (TIPs) are specifications that describe individual changes to the Tempo protocol. Each TIP documents a specific protocol upgrade including motivation, specification, and invariants.

TIPs are separate from feature-level specs (like the [TIP-20 spec](/protocol/tip20/spec) or [DEX spec](/protocol/exchange/spec)) which track the current state of each feature. When a TIP is implemented, the relevant feature specs are updated to reflect the change.

TIPs are numbered starting from 1000.

## TIP lifecycle

| Status | Description |
|--------|-------------|
| `Draft` | Initial proposal, open for feedback |
| `In Review` | Under active review by the team |
| `Approved` | Accepted for implementation |
| `Testnet` | Deployed to testnet |
| `Mainnet` | Live on mainnet |
