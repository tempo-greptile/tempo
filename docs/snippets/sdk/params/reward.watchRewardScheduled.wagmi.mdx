### onRewardScheduled

- **Type:** `(args: { funder: 0x${string}; id: bigint; amount: bigint; durationSeconds: number; }, log: Log) => void`

Callback to invoke when rewards are scheduled.

### token

- **Type:** `0x${string}`

The TIP20 token address

### args (optional)

- **Type:** `{ funder?: 0x${string} | 0x${string}[] | null; id?: bigint | bigint[] | null | undefined; } | undefined`

### fromBlock (optional)

- **Type:** `bigint`

Block to start listening from.

### onError (optional)

- **Type:** `((error: Error) => void)`

The callback to call when an error occurred when trying to get for a new block.

### poll (optional)

- **Type:** `true`

### pollingInterval (optional)

- **Type:** `number`

Polling frequency (in ms). Defaults to Client's pollingInterval config.
