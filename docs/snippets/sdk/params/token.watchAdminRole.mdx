### onRoleAdminUpdated

- **Type:** `(args: { role: 0x${string}; newAdminRole: 0x${string}; sender: 0x${string}; }, log: Log) => void`

Callback to invoke when a role admin is updated.

Address or ID of the TIP20 token.

### args (optional)

- **Type:** `{ role?: 0x${string} | 0x${string}[] | null; newAdminRole?: 0x${string} | 0x${string}[] | null | undefined; sender?: 0x${string} | 0x${string}[] | null | undefined; } | undefined`
