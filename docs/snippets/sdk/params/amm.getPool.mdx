### userToken

- **Type:** `TokenIdOrAddress`

Address or ID of the user token.

### validatorToken

- **Type:** `TokenIdOrAddress`

Address or ID of the validator token.

### account (optional)

- **Type:** `0x${string} | Account`

Account that will be used to send the transaction.

### blockNumber (optional)

- **Type:** `bigint`

The balance of the account at a block number.

### blockOverrides (optional)

- **Type:** `BlockOverrides<bigint, number>`

Block overrides for the call.

### blockTag (optional)

- **Type:** `BlockTag`

The balance of the account at a block tag.

### stateOverride (optional)

- **Type:** `StateOverride`

State overrides for the call.
