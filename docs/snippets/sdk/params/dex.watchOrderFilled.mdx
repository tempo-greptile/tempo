### onOrderFilled

- **Type:** `(args: Args, log: Log) => void`

Callback to invoke when an order is filled.

### args (optional)

- **Type:** `{ orderId?: bigint | bigint[] | null; maker?: 0x${string} | 0x${string}[] | null | undefined; } | { orderId?: bigint | bigint[] | null | undefined; maker?: 0x${string} | 0x${string}[] | null | undefined; taker?: 0x${string} | 0x${string}[] | null | undefined; } | undefined`

### fromBlock (optional)

- **Type:** `bigint`

Block to start listening from.

### maker (optional)

- **Type:** `0x${string}`

Address of the maker to filter events.

### onError (optional)

- **Type:** `((error: Error) => void)`

The callback to call when an error occurred when trying to get for a new block.

### orderId (optional)

- **Type:** `bigint`

Order ID to filter events.

### poll (optional)

- **Type:** `true`

### pollingInterval (optional)

- **Type:** `number`

Polling frequency (in ms). Defaults to Client's pollingInterval config.

### taker (optional)

- **Type:** `0x${string}`

Address of the taker to filter events.
