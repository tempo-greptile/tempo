### onApproval

- **Type:** `(args: { owner: 0x${string}; spender: 0x${string}; amount: bigint; }, log: Log) => void`

Callback to invoke when tokens are approved.

### token

- **Type:** `TokenIdOrAddress`

Address or ID of the TIP20 token.

### args (optional)

- **Type:** `{ owner?: 0x${string} | 0x${string}[] | null; spender?: 0x${string} | 0x${string}[] | null | undefined; } | undefined`

### fromBlock (optional)

- **Type:** `bigint`

Block to start listening from.

### onError (optional)

- **Type:** `((error: Error) => void)`

The callback to call when an error occurred when trying to get for a new block.

### poll (optional)

- **Type:** `true`

### pollingInterval (optional)

- **Type:** `number`

Polling frequency (in ms). Defaults to Client's pollingInterval config.
