### onTokenCreated

- **Type:** `(args: { token: 0x${string}; tokenId: bigint; name: string; symbol: string; currency: string; quoteToken: 0x${string}; admin: 0x${string}; }, log: Log) => void`

Callback to invoke when a new TIP20 token is created.

### args (optional)

- **Type:** `{ token?: 0x${string} | 0x${string}[] | null; tokenId?: bigint | bigint[] | null | undefined; } | undefined`

### fromBlock (optional)

- **Type:** `bigint`

Block to start listening from.

### onError (optional)

- **Type:** `((error: Error) => void)`

The callback to call when an error occurred when trying to get for a new block.

### poll (optional)

- **Type:** `true`

### pollingInterval (optional)

- **Type:** `number`

Polling frequency (in ms). Defaults to Client's pollingInterval config.
