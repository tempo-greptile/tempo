### onBurn

- **Type:** `(args: { sender: 0x${string}; userToken: 0x${string}; validatorToken: 0x${string}; amountUserToken: bigint; amountValidatorToken: bigint; liquidity: bigint; to: 0x${string}; }, log: Log) => void`

Callback to invoke when liquidity is removed.

### args (optional)

- **Type:** `{ sender?: 0x${string} | 0x${string}[] | null; userToken?: 0x${string} | 0x${string}[] | null | undefined; validatorToken?: 0x${string} | 0x${string}[] | null | undefined; } | undefined`

### fromBlock (optional)

- **Type:** `bigint`

Block to start listening from.

### onError (optional)

- **Type:** `((error: Error) => void)`

The callback to call when an error occurred when trying to get for a new block.

### poll (optional)

- **Type:** `true`

### pollingInterval (optional)

- **Type:** `number`

Polling frequency (in ms). Defaults to Client's pollingInterval config.

### userToken (optional)

- **Type:** `TokenIdOrAddress`

Address or ID of the user token to filter events.

### validatorToken (optional)

- **Type:** `TokenIdOrAddress`

Address or ID of the validator token to filter events.
